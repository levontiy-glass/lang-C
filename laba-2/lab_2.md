## Комплект 1: Основы работы с указателями

### Задача 1: Сложение через указатели (task01.c)

#### Постановка задачи
Получить значение переменных через указатели. Ввести два целых числа, сложить их через указатели и вывести результат.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| a | int | первое число |
| b | int | второе число |
| pa | int* | указатель на a |
| pb | int* | указатель на b |
| sum | int | сумма чисел |

#### Код программы
```c
#include<stdio.h>

int main()
{
    int a, b;
    int *pa = &a;
    int *pb = &b;
    scanf("%d %d", pa, pb);
    int sum = *pa + *pb;
    printf("%d\n", sum);
}
```

#### Результаты работы программы
```
8 9
17
```

### Задача 2: Обмен двух чисел (task02.c)

#### Постановка задачи
Изменить данные по адресу (swap). Ввести два целых числа, вывести исходные значения, затем обменённые. Обмен выполнять через указатели.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| a | int | первое число |
| b | int | второе число |
| pa | int* | указатель на a |
| pb | int* | указатель на b |
| temp | int* | временный указатель для обмена |

#### Код программы
```c
#include<stdio.h>

int main()
{
    int a, b;
    int *pa = &a;
    int *pb = &b;
    scanf("%d %d", pa, pb);
    printf("%d, %d\n", *pa, *pb);
    int *temp = pa;
    pa = pb;
    pb = temp;
    printf("%d, %d\n", *pa, *pb);
}
```


### Задача 3: Максимум через указатели (task03.c)

#### Постановка задачи
Найти максимальное из двух чисел, используя сравнение значений через указатели.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| a | int | первое число |
| b | int | второе число |
| pa | int* | указатель на a |
| pb | int* | указатель на b |
| max | int | максимальное число |

#### Код программы
```c
#include<stdio.h>

int main()
{
    int a, b;
    int *pa = &a;
    int *pb = &b;
    scanf("%d %d", pa, pb);
    int max;
    if(*pa > *pb)
    {
        max = *pa;
    }
    else
    {
        max = *pb;
    }
    printf("%d\n", max);
}
```

### Задача 4: Сумма элементов массива (task04.c)

#### Постановка задачи
Создать динамический массив, ввести количество элементов (от 2 до 999), заполнить массив, вычислить и вывести сумму всех элементов.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| n | int | количество элементов |
| sum | int | сумма элементов |
| arr | int* | указатель на динамический массив |
| i | int | счётчик цикла |

#### Код программы
```c
#include<stdio.h>
#include<stdlib.h>

int main()
{
    int n = 0, sum = 0;
    while (n <= 1 || n >= 1000)
    {
        printf("Elementov v massive: ");
        scanf("%d", &n);
        if(n <= 1 || n >= 1000)
        {
            printf("Chislo dolzhno byt bolshe 1 i menshe 1000!\n");
        }
    }
    int *arr = malloc(sizeof(int) * n);
    printf("Vvod elementov\n");
    for(int i = 0; i < n; i++)
    {
        scanf("%d", arr+i);
    }
    for(int i = 0; i < n; i++)
    {
        sum+=*(arr+i);
    }
    printf("Summa elementov: %d\n", sum);
    free(arr);
}
```

#### Результаты работы программы
```
Элементов в массиве: 5
Ввод элементов
1 2 3 4 5
Сумма элементов: 15
```

### Задача 5: Вывод массива в обратном порядке (task05.c)

#### Постановка задачи
Создать динамический массив, ввести количество элементов (от 2 до 999), заполнить массив и вывести элементы в обратном порядке.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| n | int | количество элементов |
| sum | int | сумма элементов |
| arr | int* | указатель на динамический массив |
| i | int | счётчик цикла |

#### Код программы
```c
#include<stdio.h>
#include<stdlib.h>

int main()
{
    int n = 0, sum = 0;
    while (n <= 1 || n >= 1000)
    {
        printf("Elementov v massive: ");
        scanf("%d", &n);
        if(n <= 1 || n >= 1000)
        {
            printf("Chislo dolzhno byt bolshe 1 i menshe 1000!\n");
        }
    }
    int *arr = malloc(sizeof(int) * n);
    printf("Vvod elementov\n");
    for(int i = 0; i < n; i++)
    {
        scanf("%d", arr+i);
    }
    for(int i = n-1; i >= 0; i--)
    {
        printf("%d ", *(arr+i));
    }
    printf("\n");
    free(arr);
}
```

#### Результаты работы программы
```
Элементов в массиве: 5      
Ввод элементов

```

### Задача 6: Представление числа в байтах (task06.c)

#### Постановка задачи
Вывести побайтовое представление целого числа в памяти.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| a | int | число |
| pa | int* | указатель на a |
| ppa | unsigned char* | указатель на байты числа |

#### Код программы
```c
#include<stdio.h>

int main()
{
    int a = 1234567890;
    int *pa = &a;
    unsigned char *ppa = (unsigned char*)&a;
    for(int i = (sizeof(int)/sizeof(unsigned char)) - 1; i >= 0; i--)
    {
        printf("%d ", *(ppa+i));
    }
    printf("\n");
}
```

#### Результаты работы программы
```

```

### Задача 7: Двумерный массив через указатели (task07.c)

#### Постановка задачи
Создать двумерный массив (2 строки по 3 элемента) с использованием указателей на указатели, ввести 6 элементов и вывести их в виде матрицы.

#### Список идентификаторов

| имя переменной | тип данных | описание |
| -------------- | ---------- | -------- |
| row1 | int* | первая строка |
| row2 | int* | вторая строка |
| colls | int** | указатель на массив указателей |

#### Код программы
```c
#include<stdio.h>
#include<stdlib.h>

int main()
{
    int *row1 = malloc(sizeof(int) * 3);
    int *row2 = malloc(sizeof(int) * 3);
    int **colls = malloc(sizeof(int*) * 6);
    colls[0] = row1;
    colls[1] = row2;
    for(int i = 0; i < 6; i++)
    {
        scanf("%d", (colls+i));
    }
    for(int i = 0; i < 6; i++)
    {
        printf("%d ", *(colls+i));
        if(i == 2)
        {
            printf("\n");
        }
    }
    printf("\n");
    free(colls);
    free(row2);
    free(row1);
}
```

#### Результаты работы программы
```
1 2 3 4 5 6
1 2 3 
4 5 6
```
